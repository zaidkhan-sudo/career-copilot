# CareerPilot User Flow

## Phase 1: The "Instant Value" Onboarding
**Goal:** Move the user from "Landing Page" to "Seeing a 90% Match" in under 2 minutes.

- **Auth & Entry:** One-click GitHub/Google OAuth.
- **The "Agent Hand-off":** User uploads a PDF resume. The Analyzer Agent parses skills while Scout Agent starts background scan.
- **Goal Calibration:** Single screen: "Where do you want to be in 3 years?"
- **The Magic Reveal:** Dashboard shows: "Based on your resume, I've found 12 jobs. 3 are 90%+ matches."

## Phase 2: The Passive "Hunting" Loop (Daily)
**Goal:** Keep user informed without "Job Board Fatigue."

- **Morning Intelligence Briefing:** Reporter Agent sends push/email notification
- **Dashboard Review:** Top 3 Matches with "AI Reasoning" badges
- **Quick Action:** Swipe to save or click "Prepare Materials" to trigger Writer Agent

## Phase 3: The "Precision Strike" (Active Application)
**Goal:** Generate world-class materials with zero manual work.

- **Drafting:** Writer Agent generates unique resume + cover letter
- **Human-in-the-loop (HITL):** Split-screen: Job Description vs AI-generated Resume
- **The "Ready" State:** Finalize, get PDF + tracking link, mark as "Applied"

## Phase 4: The "Preparation Arena" (Interview Prep)
**Goal:** Convert applications into offers.

- **Context Loading:** Move job to "Interview" column
- **Coach Agent Initialization:** Scrapes interview data for company
- **The Simulation:**
  - Level 1 (OA): 30-minute coding sprint
  - Level 2 (Behavioral): Voice-based Q&A
- **Feedback & Study Plan:** Identify weaknesses, generate 3-day study plan

## Phase 5: The "Evolution" Loop (The Brain)
**Goal:** Get smarter with every rejection or success.

- **Outcome Entry:** User marks job as "Rejected"
- **Root Cause Analysis:** Dropdown (Technical fail, Ghosted, Cultural fit)
- **Agent Recalibration:** Adjust strategies based on outcomes
- **Strategy Update:** Morning briefing reflects adjusted parameters
